算法第一篇

世界之大，算法之美，你终会理解O(n^2)、O(n*logn)、O(n)之间差别。

二分法查找的时间复杂度O(logn)

一组（n个）有序数列中找所要的key值，返回位置。每次与n/2位置数比较，知道找到或n==1, 停止检索。
则最坏情况（即最后到1的）假设二分操作了x次（则O(x)），n*(1/2)^x=1;推算x=logn(以2为底)。其实logn、lgn、lnn是无差的，因为他们的比值是一个常量。

将10亿个ipV4地址排序

一个ipV4地址占内存32位，ipV4地址的总个数为2^32个。
将ipV4地址转化为整数，建立一个长度2^32的数组arr，遍历那10亿个ipV4地址整数，将ipV4整数-1对应arr下标，将值置为1,以此方式遍历完10亿整数。之后遍历数组arr，找元素值为1的下标+1，然后转化为ipV4地址格式完成排序。时间复杂度O(n)。
类似的，电话号码排序，身份证好排序。有重复就在那个位置值+1，最后查不是null的就行。

KMP算法

参考：http://www.cnblogs.com/c-cloud/p/3224788.html
http://blog.csdn.net/yutianzuijin/article/details/11954939/
1.O(nm)解：两层循环遍历，外层1-n原字符串str（称大串）一个一个往后移动，内层1-m匹配字符串s（称小串）一个一个与大串字符比较，直到全部匹配成功，不成功跳出，小串后移一个字符。
2.O(n+m)解：与1算法唯一区别仅在于如果匹配不成功，小串后移k字符（k>=1），而且下一轮内层遍历比较从刚刚匹配失败字符位置开始继续比较遍历。

然后需要求k值，设头字符到匹配失败字符前的字符为子串。
引入一个概念：部分匹配值一个字符串的前缀与后缀相同的最大长度（前缀和后缀都是字符串本身），如：ab12ab是2,cbcbc是3，1234是不存在的就是0，aaaa对应是3。
想啊，小串匹配失败前的子串（已匹配长度）同时在大串中存在，则小串移动到前缀与大串对应的后缀重合，如果相同的前后缀不存在就移动到匹配失败字符即可。*k=已匹配字符长度-部分匹配值之后的匹配，大串后缀与小串前缀对应匹配，所以大串匹配索引不变，小串索引是前缀的下个字符（j=next[j-1]）。
创建数组next[m]存放小串所有子串对应的部分匹配值。运用递归计算。

int[] next(String sub){
    int m = s.length();
    int next[m];
    int k=0;//部分匹配值，next[i],即最大前后缀长度
    next[0]=0;//对于前一个字符
    for(int i=1;i<m;i++){//求每个字符对应的next[]值，即前i个字符的最大前后缀长度
        //只有前i-1字符有k>=1个最大前后缀匹配，才能进这个while
        while(k>0 && sub.charAt(i) != sub.charAt(k)){//s[k]表示 前缀 后的一个字符
            //加一个字符后k++失败，即aba？加个字符变abaa？如果是abab就跳出了。（飞升失败。）
            k=next[k-1];//next[k-1]中k是前i-1字符的最大前、后缀，k-1对应的next是前缀的最大前后缀（这个前缀也是前i-1字符的后缀，只不过不是最大），然后进行之后的比较。例如ABADABA->ABADABAB就是2了
        }
        if(sub.charAt(i) == sub.charAt(k)){
            k++;
        }
        next[i] = k;
    }
    return next;
}

public static int kmpZ(String str,String s){
    int index=-1;//求得的大串的索引
    if((str==null||str.length()==0)||(s==null||s.length()==0)){
        System.out.println("主字符串和匹配字符串不能为空！");
        return index;
    }
    int[] next = next(s);
    int j = 0;//指示小串的下标
    for(int i=0;i<str.length();i++){
        while(j>0 && str.charAt(i) != s.charAt(j)){
            j=next[j-1];//解释看上面
        }
        if(str.charAt(i) == s.charAt(j)){
            j++;
        }
        if(s.length()==j){//注意j从0开始的
            index = i-(j-1);
            return index;//这句也是必须的
        }
    }
    return index;
}

字符串相关

1.（旋转字符串）给定一个字符串，要求把字符串前面的若干个字符移动到字符串的尾部，如把字符串“abcdef”前面的2个字符’a’和’b’移动到字符串的尾部，使得原字符串变成字符串“cdefab”
要求：时间复杂度为 O(n)，空间复杂度为 O(1)

变化过程：abcdef中分别将‘ab’、‘cdef’做自身反转得到‘bafedc’，然后整个做自身反转得到‘cdefab’
ps：StringBuilder.reverse()可以反转字符串。
2.（反转句子型字符串，单词不反转）即以单词为单位反转，保留空格，如”I’am a student”->”student a I’am”

变化过程：整体反转得到”tneduts a ma’I”,然后单个单词反转得到”student a I’am”
ps: StringBuilder的resverse()方法和append()方法，使用String的split()方法分割字符串。
3.字符串算数表达式求值（如12+(23 3-56+7)(2+90)/2）
先将字符串转化为List，将中缀表达式转化为后缀表达式，计算后缀表达式，比较运算符等级。

import java.util.ArrayList;  
import java.util.Stack;   
public class Calculate {     
    /** 
     * 将字符串转化成List 
     * @param str 
     * @return 
     */  
    public ArrayList<String> getStringList(String str){  
        ArrayList<String> result = new ArrayList<String>();  
        String num = "";  
        for (int i = 0; i < str.length(); i++) {  
            if(Character.isDigit(str.charAt(i))){  
                num = num + str.charAt(i);  
            }else{  
                if(num != ""){  
                    result.add(num);  
                }  
                result.add(str.charAt(i) + "");  
                num = "";  
            }  
        }  
        if(num != ""){  
            result.add(num);  
        }  
        return result;  
    }   
    /** 
     * 将中缀表达式转化为后缀表达式 
     * @param inOrderList 
     * @return 
     */  
    public ArrayList<String> getPostOrder(ArrayList<String> inOrderList){      
        ArrayList<String> result = new ArrayList<String>();  
        Stack<String> stack = new Stack<String>();  
        for (int i = 0; i < inOrderList.size(); i++) {  
            if(Character.isDigit(inOrderList.get(i).charAt(0))){  
                result.add(inOrderList.get(i));  
            }else{  
                switch (inOrderList.get(i).charAt(0)) {  
                case '(':  
                    stack.push(inOrderList.get(i));  
                    break;  
                case ')':  
                    while (!stack.peek().equals("(")) {  
                        result.add(stack.pop());  
                    }  
                    stack.pop();  
                    break;  
                default:  
                    while (!stack.isEmpty() && compare(stack.peek(), inOrderList.get(i))){  
                        result.add(stack.pop());  
                    }  
                    stack.push(inOrderList.get(i));  
                    break;  
                }  
            }  
        }  
        while(!stack.isEmpty()){  
            result.add(stack.pop());  
        }  
        return result;  
    }    
    /** 
     * 计算后缀表达式 
     * @param postOrder 
     * @return 
     */  
    public Integer calculate(ArrayList<String> postOrder){  
        Stack stack = new Stack();  
        for (int i = 0; i < postOrder.size(); i++) {  
            if(Character.isDigit(postOrder.get(i).charAt(0))){  
                stack.push(Integer.parseInt(postOrder.get(i)));  
            }else{  
                Integer back = (Integer)stack.pop();  
                Integer front = (Integer)stack.pop();  
                Integer res = 0;  
                switch (postOrder.get(i).charAt(0)) {  
                case '+':  
                    res = front + back;  
                    break;  
                case '-':  
                    res = front - back;  
                    break;  
                case '*':  
                    res = front * back;  
                    break;  
                case '/':  
                    res = front / back;  
                    break;  
                }  
                stack.push(res);  
            }  
        }  
        return (Integer)stack.pop();  
    }   
    /** 
     * 比较运算符等级 
     * @param peek 
     * @param cur 
     * @return 
     */  
    public static boolean compare(String peek, String cur){  
        if("*".equals(peek) && ("/".equals(cur) || "*".equals(cur) ||"+".equals(cur) ||"-".equals(cur))){  
            return true;  
        }else if("/".equals(peek) && ("/".equals(cur) || "*".equals(cur) ||"+".equals(cur) ||"-".equals(cur))){  
            return true;  
        }else if("+".equals(peek) && ("+".equals(cur) || "-".equals(cur))){  
            return true;  
        }else if("-".equals(peek) && ("+".equals(cur) || "-".equals(cur))){  
            return true;  
        }  
        return false;  
    }   
    public static void main(String[] args) {  
        Calculate calculate = new Calculate();  
        String s = "12+(23*3-56+7)*(2+90)/2";  
        ArrayList result = calculate.getStringList(s);  //String转换为List  
        result = calculate.getPostOrder(result);   //中缀变后缀  
        int i = calculate.calculate(result);   //计算  
        System.out.println(i);  
    }    
}

4.字符串相似度（编辑距离）
许多程序会大量使用字符串。对于不同的字符串，我们希望能够有办法判断其相似程度。我们定义了一套操作方法来把两个不相同的字符串变得相同，具体的操作方法为：
修改一个字符（如把“a”替换为“b”）。
增加一个字符（如把“abdd”变为“aebdd”）。
删除一个字符（如把“travelling”变为“traveling”）。
比如，对于“abcdefg”和“abcdef”两个字符串来说，我们认为可以通过增加/减少一个“g“的方式来达到目的。上面的两种方案，都仅需要一次操作。把这个操作所需要的次数定义为两个字符串的距离，给定任意两个字符串，写出一个算法来计算出它们的距离
距离，即两个字符串，由一个转成另一个所需最少的编辑次数。
设数组dp[i][j]的含义：str2(“0->i”)通过操作变成str1(“0->j”)的编辑距离
考虑以下三种情况
str2(“0->i”)与str1(“0->j-1”)相同，在末尾增加一个str1[j]字符，故此时dp[i][j] = dp[i][j-1] + 1；
str2(“0->i-1”)与str1(“0->j”)相同，在末尾删除一个str2[i]字符，故此时dp[i][j] = dp[i-1][j] + 1；
str2(“0->i-1”)与str1(“0->j-1”)相同，根据str2[i]与str1[j]的相等情况，分别考虑。str2[i]等于str1[j]时，无需任何操作，故此时dp[i][j] = dp[i-1][j-1];str2[i]不等于str1[j]时，替换str2[i]字符为str1[j]字符，故此时dp[i][j] = dp[i-1][j-1] + 1；
根据上面的讨论，我们可以看到dp[i][j]的值依赖它左上角的3个dp值。在程序的开始阶段，我们应首先计算得到最不需要依赖的dp值，也就是整个dp二维数组中的第一行与第一列。

for(int i = 0; i < lenA; i++) dp[i][0] = i;
for(int j = 0; j < lenB; j++) dp[0][j] = j;
dp[0][0] = 0;
